# Unity Makeup Game

## Описание
2D игра, где игрок может:
- Убирать прыщи кремом
- Наносить тени (2 цвета)
- Наносить помаду
- Полностью стирать макияж спонжем

## Технические моменты
- Unity 6.3
- DragTool.cs — универсальный инструмент для всех инструментов
- BookTabs.cs — переключение вкладок
- FaceController.cs — управление состоянием лица
- DragToolManager.cs — контролирует активный инструмент
- Плавное движение инструментов и привязка к курсору
